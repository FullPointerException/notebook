* Emacs
** Navigation
*** Opening a file
C-x C-f opens find file
*** File navigation
| Behavior  | Forward | Reverse |
|-----------+---------+---------|
| Line      | C-n     | C-p     |
| Word      | M-f     | M-b     |
| Character | C-f     | C-b     |
|           |         |         |
*** Buffers 
C-x C-b opens buffer menu in new window
C-x o switches to that window
*** Windows
C-x o switches windows
C-x 0 kills the current windo
C-x 3 opens a second window vertically
